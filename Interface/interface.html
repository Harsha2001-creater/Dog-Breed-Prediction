Dog Breed Classification Project

An AI-powered system to predict the breed of a dog from an image

About the Project

This project aims to develop an AI model capable of accurately predicting the breed of a dog based on an input image. The model utilizes convolutional neural networks (CNNs) trained on a dataset of dog images.
